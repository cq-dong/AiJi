# AiJi 参赛 500 字符文案

## 1. 作品简介（500字符以内）

日常值得记的内容形态各异——灵感闪念、项目进展、生活备忘、待办提醒——传统笔记靠手动归类，类别僵化且维护成本高；通用AI又不懂"你"，推荐千篇一律，而日记类应用又强制每日一记，无法适配碎片化场景。AiJi是全模态AI随手记：文本、语音、图片可在同一条目中混合输入，语音经Paraformer实时转写为文字，图片由VLM理解并生成描述，多模态信息融合后统一进入AI处理管线。LLM从每条内容自动涌现类别、提取标签、识别情绪侧面与提醒意图，不预设固定分类体系，类别随内容积累自然涌现与演化，用户可随时合并、重命名或新增来策展自己的知识地图。AI还自动生成周期聚合，帮你回看记录脉络。同时支持显式教AI规则（"AI记忆"），如"螺蛳粉归美食类"或"凡是提到老板的消息都标记为工作"，教一次即贯穿后续所有分类与问答始终受用，越用越懂你。自然语言对话检索，双轮LLM带思维链推理，先规划检索策略再生成回答，有据可引、可追溯。数据本地优先，用户数据存于设备本地，密钥不下发，AI调用经自有代理转发。已交付APK与PWA双端，移动端可直接使用。

## 2. 作品方向描述（500字符以内）

AiJi属于应用创新方向，是面向个人知识管理场景的AI原生应用，将大模型能力深度融入日常记录的全流程——从采集、理解、组织到检索与回溯，每一步都有AI参与，而非简单地在传统笔记外挂一个AI助手。创新有三：其一，涌现分类——不同于传统笔记的固定分类体系，LLM从每条内容自动发现类别，类别随内容积累自然涌现与演化，用户可策展合并、重命名或新增，实现"分类长出来而非框进去"。其二，AI记忆——用户显式教AI规则（如"螺蛳粉归美食类"），规则持久生效、贯穿分类与问答，实现"教一次就懂你"的个性化体验，越用越懂。其三，对话检索——自然语言问库，双轮LLM带思维链推理，先规划检索策略再生成回答，有据可引、可追溯。技术上，端口/适配器分层架构使Domain零平台耦合，同一代码三态验证（PWA/Android/服务端），鸿蒙化只需换适配器。隐私上本地优先，用户数据存于设备本地，密钥不下发，AI调用经自有代理转发，数据可控可迁移，兼顾智能与隐私。AiJi让记录从负担变为享受，让每个人拥有一座随记随长、越用越懂自己的个人知识库。
